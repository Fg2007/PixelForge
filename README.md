# PixelForge Studios

Landing page moderna, responsiva e interativa para um estudio ficticio de curadoria gamer chamado PixelForge Studios. O projeto foi criado para praticar HTML5, CSS3, JavaScript, responsividade, organizacao de arquivos e preparacao para futura publicacao no GitHub Pages.


## Tecnologias utilizadas

- HTML5 com tags semanticas: `header`, `main`, `section` e `footer`
- CSS3 com Flexbox, CSS Grid, Media Queries, transicoes, sombras e efeitos neon
- JavaScript para rolagem suave, validacao de formulario, filtro, carrossel, tela de loading, tema claro/escuro com Local Storage e musica de fundo sintetizada
- Google Fonts: Press Start 2P, Roboto e Montserrat
- YouTube Embed para o trailer destaque

## Como executar o projeto

Abra o arquivo `index.html` no navegador. Tambem e possivel usar a extensao Live Server no Visual Studio Code.

## Funcionalidades

- Menu de navegacao funcional
- Menu hamburguer responsivo
- Catalogo com 5 jogos reais: Honkai: Star Rail, Elden Ring, Hades, Hollow Knight e Stardew Valley
- Carrossel de jogos
- Filtro por genero
- Trailer da Temporada com YouTube Embed
- Depoimentos da comunidade
- Formulario de contato com validacao
- Alternancia entre modo claro e modo escuro
- Preferencia de tema salva no Local Storage
- Tela de loading inicial
- Efeitos neon animados e transicoes suaves
- Musica de fundo ativada por botao
